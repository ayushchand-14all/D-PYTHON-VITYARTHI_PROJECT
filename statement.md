## Project Title: Mastermind
### Description
Mastermind is an interactive game that challenges users to guess a secret 4-digit number. The user has a maximum of 12 attempts to correctly guess the number. 

### Game Rules
- The game generates a random 4-digit secret key.
- The user must guess the secret key within 12 attempts.
- If the user guesses the correct number within the allowed attempts, they win; otherwise, the Computer wins.

### Objectives
- To provide an engaging and fun way for users to test their guessing skills.
- To enhance programming skills through the development of a simple game.
